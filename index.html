About this tool
How it works: This app updates automatically every morning at 8am UK time. A scheduled script calls the Anthropic Claude API with web search enabled to find currently open grants matching your charity's profile. Results are saved and displayed in the "Today's opportunities" tab. The main funder list is curated and reviewed periodically. The AI Search tab lets you ask specific questions in real time. Built using GitHub Actions (scheduler), Vercel (hosting), and the 360Giving open API for historical grant data.
